          A Uniform Resource Locator (URL or web address) has a scheme that includes an information about what it is: a file or a news.

      CSS
      1. Name the components of a CSS rule property
          Selector and in curly brackets is the declaration with a property:
          with value and semicolon you can define as many declarations as you want.
      2. Use combinations properly: direct child, descendant, adjacent sibling, siblings 
          o A descendant connects all elements that are descendants of a specified element
          o An adjacent sibling selector selects all elements, that are adjacent siblings of a specified element.
            It is important, that they have the same parent because they follow them immediately. 
          o The general sibling selector selects all elements that are siblings of a specified element. 
      3. Specify colors by color names and in hexadecimal notation
          There are different ways to declare a color.          
          o By name: <h1 style="color:Tomato">Hello World</h1>
          o By hex: <h1 style="color:#8000ff">Hello World</h1>   
      4. Declarations for color, background color
          Declaration for color see above example
          Declaration for background-color:
          o By name: <h1 style="background-color:Tomato">Hello World</h1>
          o By hex: <h1 style="background-color:#8000ff">Hello World</h1>      
      5. Pseudo classes for the <a> element
          You can make for nearly everything pseudo classes that can do some useful things e.g.
          the link will change the color when the cursor goes over it.  
      6. Declaration for background image
          You can specifiy a background for the body and you also declare a background for a paragraph as well.
          A background image can be declared in a css area, like a paragraph. 
           <style>
            body {
                background-image: url("img_tree.png");
                background-repeat: no-repeat;
                background-position: right top;
                margin-right: 200px;
                background-attachment: fixed;
            }
           </style> 
     7. Declaration for text alignment (left,right,center,justify)
         Is used to define the horizontal alignment of text.
     8. Declaration for text decoration (overline, underline, line-through)
         o Text-decoration: non is used to remove decorations from a text
         o Text-decoration: overline or line-through or underline is used to decorate a text.     
     9. Declaration for text transformation (uppercase, lowercase, capitalize)
         This is used to specify, how the (first)letter in a word should look like, e.g.
          <style>          
           p.capitalize {
               text-transform: capitalize;
           }
          </style>
          < class="capitalize">HelloWorld</p>
    10. Identify the difference between serif and sans-serif fonts
         Serif fonts haves small lines at the end in some characters, a typical serif font us "Times New Roman".
         Non-serif font do have clear characters, without any lines at the end, e.g. Arial or Verdana.
    11. Declarations for font families
         Font families is a specific font family, e.g. Arial, Arial Black or Arial Narrow.
    12. Declarations for font style normal and italic
         This defines if a paragraph is written in normal style or in italic
          <style>
           p.normal {
               font-style: normal;
           }
           p.italic {
               font-style: italic;
           }
          <style>
    13. Declarations for font sizes
         This declaration sets the size of the text. There are three different ways to declare it:
         one of them is to declare them by pixel, e.g. font-size: 20px; the second option is to declare them by “Em”.
         The advantage of “Em”s is, that the browser can scale them, e.g. font-size: 1.875em;
         The third option is the declaration by Percent, e.g. font-size: 120%;
    14. Declarations for font weights
         This specifies if the font is normal, light, bold.
